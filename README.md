\textbf{Data Science Project Setup with CookieCutter and DVC}

\section*{Project Organization}

\textbf{Objective:}
The objective of this assignment is to understand the setup and usage of CookieCutter for structuring a data science project and DVC (Data Version Control) for managing data and model versioning efficiently.

\textbf{Project Structure:}

\begin{itemize}
\item \textbf{data/} - Stores raw and processed data.
\item \textbf{docs/} - Documentation for the project.
\item \textbf{models/} - Stores trained models.
\item \textbf{notebooks/} - Jupyter notebooks for exploration.
\item \textbf{reports/} - Generated reports and findings.
\item \textbf{src/} - Source code for the project (scripts, pipelines, etc.).
\item \textbf{tests/} - Unit tests for the project.
\end{itemize}
